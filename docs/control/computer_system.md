### 数据通路
**执行部件间传输信息的路径。**

建立由控制信号控制，受时钟驱动。

数据通路描述了信息从什么地方开始，中间经过哪个寄存器或多路开关，最后传送到哪个寄存器。

数据通路中专门进行数据运算的部件称为执行部件或功能部件；数据通路由控制部件控制。

数据通路的功能是实现 CPU 内部的运算器与寄存器及寄存器之间的数据交换。
